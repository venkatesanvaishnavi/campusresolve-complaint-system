# CampusResolve – Automated Complaint Management System

CampusResolve is a full-stack complaint management and deployment automation platform developed for managing campus complaints efficiently.

## Features
- Student and Authority Login
- Complaint Submission and Tracking
- Authority Status Management
- REST API Backend
- MySQL Integration
- Ansible Automation
- Secure Variable Management using Ansible Vault

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
- Ansible
- Ubuntu WSL

## DevOps Features
- Automated package installation
- Service management
- Templates and handlers
- Inventory management
- Ansible tags
- Vault encryption
